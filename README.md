# smart-scissors-image-segmentation
# Intelligent Scissors Image Segmentation Tool

基于改进 Dijkstra 算法的智能剪刀图像分割工具。项目通过图像梯度构建路径代价图，并结合实时 LiveWire 交互，实现对目标物体边缘的半自动轮廓提取、闭合选区、抠图预览与 PNG 结果保存。

## 项目简介

智能剪刀是一类交互式图像分割方法：用户通过鼠标指定少量种子点，程序根据图像边缘强度自动计算两点之间最贴合边缘的路径，从而降低手动抠图成本。

本项目使用 Java Swing 实现图形界面，核心算法基于 Dijkstra 最短路径搜索，并针对图像分割场景加入方向一致性、路径平滑、曲率约束、弱边缘惩罚、细边缘检测和边缘吸附等改进，使路径能够更稳定地贴合目标边界。

## 功能特性

- 支持导入常见图片格式：`jpg`、`jpeg`、`png`、`gif`
- 支持彩色图与灰度图的边缘检测和路径搜索
- 支持鼠标实时 LiveWire 路径预览
- 支持左键添加锚点、右键撤销路径段、中键闭合或新建轮廓
- 支持靠近起点时的自动闭合提示
- 支持执行抠图并导出透明背景 PNG
- 支持边缘吸附，降低鼠标点击误差
- 支持路径平滑、插值和跳点修复，减少路径断裂与锯齿

## 算法思路

### 1. 图像预处理

项目使用 `Preproc` 类完成图像梯度计算：

- 对灰度图进行直方图均衡化，增强对比度
- 使用 Sobel 算子计算水平和垂直方向梯度
- 使用 Laplacian 算子辅助增强边缘响应
- 对彩色图分别计算 RGB 多通道梯度，并融合最大梯度与平均梯度
- 根据梯度强度构建代价图：边缘越强，路径代价越低

### 2. 改进 Dijkstra 路径搜索

`PathFinder` 类在传统 Dijkstra 最短路径基础上加入多种图像分割约束：

- **边缘强度代价**：优先选择梯度强、边缘明显的像素点
- **方向一致性代价**：鼓励路径沿着边缘方向连续前进
- **平滑度代价**：减少路径频繁转折
- **曲率代价**：降低突变路径出现概率
- **弱边缘惩罚**：避免路径穿过非边缘区域
- **细边缘检测**：提升对细小边界的识别能力
- **目标方向约束**：引导路径向终点方向收敛

最终路径会经过插值、均值滤波和跳点修复，使轮廓更加连续、平滑。

### 3. 边缘吸附

`EdgeSnapUtil` 类实现鼠标点位的边缘吸附：

1. 在鼠标当前位置附近的局部区域内搜索，默认半径为 `8`
2. 计算局部平均梯度值
3. 筛选梯度强度高于平均值 `1.2` 倍的候选点
4. 判断候选点是否为局部最大值
5. 返回最优像素点作为新的 LivePath 起点或终点

该功能可以让用户点击位置自动吸附到附近强边缘，提升交互精度。

## 操作说明

启动程序后：

- 点击 **选择图像** 导入图片
- 鼠标左键：添加当前路径锚点
- 鼠标移动：实时预览从当前锚点到鼠标位置的边缘路径
- 鼠标右键：撤销上一个路径段
- 鼠标中键：闭合当前轮廓或开始新轮廓
- 靠近起始点时：自动提示闭合
- 点击 **执行剪切**：根据闭合轮廓生成抠图结果
- 点击 **保存**：导出 PNG 图片

## 运行方式

### 环境要求

- JDK 8 或更高版本
- 支持 Swing 的桌面环境

### 使用命令行运行

在项目根目录下执行：

```bash
javac -encoding UTF-8 -d out src/*.java
java -cp out IntelligentScissorsGUI
```

如果使用 Windows PowerShell，也可以执行：

```powershell
javac -encoding UTF-8 -d out .\src\*.java
java -cp out IntelligentScissorsGUI
```

### 使用 IntelliJ IDEA 运行

1. 使用 IntelliJ IDEA 打开项目目录
2. 将 `src` 标记为源码目录
3. 运行 `IntelligentScissorsGUI.java` 中的 `main` 方法

## 核心亮点

- 在路径代价函数中融合梯度、方向、曲率和平滑度等多维信息

## 效果说明

该工具适合用于边缘较清晰的目标抠图任务。用户只需要沿目标轮廓点击少量锚点，系统即可自动搜索并生成贴合边缘的路径。对于彩色图、灰度图、细边缘和局部弱边缘场景，项目通过多项代价函数和路径后处理策略提升了适应性。
